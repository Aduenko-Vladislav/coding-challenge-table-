## Instructions
1. Clone this repository:
   `git clone <repository-url>`
2. Navigate into the project directory:
   `cd <project-directory>`
3. Install the dependencies:
   `npm install`
4. Start the development server:
   `npm run`
5. Open your browser and go to:
   http://localhost:3000
